# VAST - Distribution Wheel Files

This repository contains distribution wheels for VAST related code created by Jesse Thaloor. 

Here are the current released wheel packages. 

## VAST S3
Python package for S3 operations. This should also work for other S3 providers as well.

https://github.com/jesseVast/vast/releases/download/vasts3-1.1.19/vasts3-1.1.19-py3-none-any.whl

## VAST DBManager
Python package for connecting to VAST DataBase with support for Trino and native query engine. 

https://github.com/jesseVast/vast/releases/download/vastdbmanager-v2.0.1/vastdbmanager-2.0.1-py3-none-any.whl

## AI Fuel
Python package to connect to AI providers for Embedders, ReRankers, LLMs, VLMs and VSSs. 

https://github.com/jesseVast/vast/releases/download/aifuel-v0.3.4/aifuel-0.3.4-py3-none-any.whl

# Video Tools
Ffmpeg based python library to run ffmpeg processes. 

https://github.com/jesseVast/vast/releases/download/videotools-v0.2.0/videotools-0.2.0-py3-none-any.whl

# Vast Istore
VastEngine Pipeline interfunction Store that automatically use the right temp storage location based on data size.

https://github.com/jesseVast/vast/releases/download/vastistore-v1.1.17/vastistore-1.1.17-py3-none-any.whl
