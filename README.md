# VAST - Distrubution Wheel Files

This repository contains distribution wheels for VAST related code created by Jesse Thaloor. 

Here are the current released wheel packages. 

## VAST S3
Python package for S3 operations. This should also work for other S3 providers as well.

https://github.com/jesseVast/vast/releases/download/vasts3-1.1.18/vasts3-1.1.18-py3-none-any.whl

## VAST DBManager
Python package for connecting to VAST DataBase with support for Trino and native query engine. 

https://github.com/jesseVast/vast/releases/download/vastdbmanager-v2.0.0/vastdbmanager-2.0.0-py3-none-any.whl

## VAST IStore
Python package for use with VAST DataEngine that enables wrapping inter function pipeline messages to be stored in-pipeline, S3 or Redis based on the message sizes.
